# 06 - Quiz Program 

Um programa que faz 7 perguntinhas sobre capitais de paises e vai comparar as strings escritas pelo usuario e os nomes salvos anteriormente pelo desenvolvedor do sistema. 
A pegadinha è que se usando o ".lower()" temos que fazer uma coisa exata, porque apesar de ele entender tudo como minusculo e poder ignorar esse tipo de erro, uma letrinha 
errada e ja era o seu jogo, pode ser considerado erro se nao tiver exatamente a mesma resposta gravada tipo "Rome" è diferente de "Roma" e nao sera computado como um acerto 
para o usuario do software. Talvez um tratamento de erros combinado com tratamento de strings seja possivel confirmar isso, por exemplo se a gente conseguir comparar letra a 
letra dentro de uma string onde apenas uma seja diferente, podemos por exemplo colocar um lembrete de que o programa esta rodando no idioma tal e a palavra com uma letra 
diferente ja sera considerada errada. Poderia ser interessante ter esse tipo de recurso. 

# 07 - QR Code Generator

Gera um QR Code para um atalho para qualquer site por exemplo que voce queira. Podemos usar isso no programa hacker para pegar as informaçoes das pessoas. 
Um "ataque" assim seria colocar esse QRCode levando para um site qualquer sem aviso, que captasse os dados de cada usuario que entrasse ali e fizesse um grande banco de dados. 
Fazendo por exemplo a impressao em massa desse QR Code e espalhando por cidades, colocando adesivos por exeplo em pontos turisticos para espalhar e captar os dados de pessoas
de diversos pontos do mundo que ali passarem.
